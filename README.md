# SmartDataCrafter - Customizable Dataset Generator

SmartDataCrafter allows devs to define smart schemas and produce randomized, structured data for tests and prototypes.

## Features
- JSON / CSV / SQL output  
- Custom fields and presets  
- Clean CLI tool
